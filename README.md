### 👋 About

- 🎨 **Designer + 💻 Frontend developer in one** — I build products end-to-end, from a Figma mockup to responsive React in production.
- ⚡ Working since **2022** on product SaaS platforms (AI agents, mining farms, data & payouts): complex dashboards, real-time data, charts, maps, design systems.
- 🧩 My edge — **design + code**: I see an interface as a designer and ship it as a developer, with nothing lost in translation.
- 🧠 I use AI as a daily tool — local models, image/video generation, prototyping.
- 📍 Based in Tomsk · open to remote.
